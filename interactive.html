Interactive Demos

  • Byte Pair Encoding Visualization

    This visualization aims to demonstrate the tokenization as performed by the byte-pair-encoding algorithm, as originally described here. This encoding process can change drastically based on the corpus used to train the merges and the amount of merge operations, thus this experience allows for users to select from one of 3 corpora from varying genres to see how the tokenization of their inputted text changes as well as the maximum number of merges to use.

    View Code

  • GloVe Embedding Sandbox

    This is a sandbox playground for testing how GloVe embeddings can be added and subtracted. The embeddings are 10-dimensional and calculated following the methods outlined here. The corpus used was the article text from a partial English Wikidump which contained over 5M unique tokens, of which only 900k are retained.

  • Conway's Game of Life

    This is an interactive demo of Conway's Game of Life run entirely in the browser. One can explore different structures and how they grow and change according to a very simple set of rules.
